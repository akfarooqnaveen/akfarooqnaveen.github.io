Verify if modem installation is successful - Model M7215

This document describes steps to verify if modem installation has been successful.

Verification Steps

  1. Do an API call: Do an API call to http://localhost:8889/test/
  2. Check bandwidth: In the API response, check bandwidth value is above 100 Mbps
  3. Check modem status: In the API response, check the registration status. This should be success
